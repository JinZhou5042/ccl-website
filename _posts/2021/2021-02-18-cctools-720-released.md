<p>The Cooperative Computing Lab is pleased to announce the release of version 7.2.0 of the Cooperative Computing Tools including Parrot, Chirp, JX, Makeflow, WorkQueue, and other software.<br/><br/>The software may be downloaded here:<br/><a href="http://ccl.cse.nd.edu/software/download">http://ccl.cse.nd.edu/software/download</a><br/><br/>This is a minor release with some new features and bug fixes. Among them:<br/><br/></p>

<ul style="text-align: left;"><li>[Batch] Improved gpu handling with HTcondor. (Douglas Thain)</li><li>[WorkQueue] Resource usage report with work_queue_status per task and worker. (Thanh Son Phung)</li><li>[WorkQueue] Improved gpu handling. (Douglas Thain, Tim Shaffer)</li><li>[WorkQueue] Assign tasks to specific gpus with CUDA_VISIBLE_DEVICES. (Douglas Thain)</li><li>[Makeflow] Several fixes for sge_submit_makeflow. (Ben Tovar)</li></ul>

<p>Thanks goes to the contributors for many features, bug fixes, and tests:<br/><br/></p>

<ul style="text-align: left;"><li>Ben Tovar</li><li>Douglas Thain</li><li>Nathaniel Kremer-Herman</li><li>Thanh Son Phung</li><li>Tim Shaffer</li></ul>

<p>Please send any feedback to the CCTools discussion mailing list:<br/><br/><a href="http://ccl.cse.nd.edu/community/forum">http://ccl.cse.nd.edu/community/forum</a><br/><br/>Enjoy!<br/><br/></p>
