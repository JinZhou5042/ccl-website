<h3 class="post-title entry-title" style="background-color: white; font-family: 'Trebuchet MS', Trebuchet, Verdana, sans-serif; font-size: 22px; font-weight: normal; margin: 0.75em 0px 0px;">
<span style="font-size: 13.1999998092651px; line-height: 1.4;">Haiyan Meng presented our work on Preservation Framework for Computational Reproducibility at the International Conference on Computational Science (ICCS) in Reykjavik, Iceland. This is a collaborative work between University of Notre Dame and University of Chicago for the <a href="https://daspos.crc.nd.edu/">DASPOS</a> project both of these two universities are working on.</span></h3>

<p><span style="font-size: 13.1999998092651px; line-height: 1.4;"><br/></span></p>

<p><span style="font-size: 13.1999998092651px; line-height: 1.4;">The preservation framework proposed in this paper includes three parts: </span></p>

<p><ul>
<li><span style="font-size: 13.1999998092651px; line-height: 1.4;">First, how to use light-weight application-level virtualization techniques to create a reduced package which only includes all the necessary dependencies; </span></li>
<li><span style="font-size: 13.1999998092651px; line-height: 1.4;">Second, how to organize the data storage archive to preserve these packages; </span></li>
<li><span style="font-size: 13.1999998092651px; line-height: 1.4;">Third, how to distribute applications through standard software delivery mechanisms like Docker and deploy applications through flexible deployment mechanisms such as Parrot, PTU, Docker, and chroot.</span></li>
</ul></p>

<p><span style="font-size: 13.1999998092651px; line-height: 1.4;"><br/></span></p>
