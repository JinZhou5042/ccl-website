The Cooperative Computing Lab is pleased to announce the release of version 5.2.0 of the Cooperative Computing Tools including Parrot, Chirp, Makeflow, WorkQueue, SAND, All-Pairs, Weaver, and other software.

<br/>

<br/>

The software may be downloaded here:

<br/>

<br/>

<a href="http://www.cse.nd.edu/~ccl/software/download">http://www.cse.nd.edu/~ccl/software/download</a>

<br/>

<br/>

This minor release considers the following issues from version 5.1.0:

<br/>

<br/>

<ul>
<li>[Chirp]     Fix mkdir python binding. (Ben Tovar)</li>
<li>[Chirp]     Adds 'ln' for file links. (Nate Kremer-Herman)</li>
<li>[Chirp/Confuga] Kill a job even on failure. (Patrick Donnelly)</li>
<li>[Debug]     Fix log rotation with multiple processes. (Patrick Donnelly)</li>
<li>[Makeflow]  Better support for Torque and SLURM for XSEDE. (Nick Hazekamp)</li>
<li>[Parrot]    Fix bug where cvmfs alien cache access was sequential. (Ben Tovar)</li>
<li>[Parrot]    Allow compilation with iRODS 4.1. (Ben Tovar)</li>
<li>[WorkQueue] Improvements to statistics when using foremen. (Ben Tovar)</li>
<li>[WorkQueue] Fix bug related to exporting environment variables. (Ben Tovar)</li>
<li>[WorkQueue] Task sandboxes where not being deleted at workers. (Ben Tovar)</li>
</ul>
