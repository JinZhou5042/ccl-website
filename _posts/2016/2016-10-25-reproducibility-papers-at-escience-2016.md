<span style="font-family: inherit;">CCL students presented two papers at the <a href="http://escience-2016.idies.jhu.edu/">IEEE 12th International Conference on eScience</a> on the theme of reproducibility in computational science:</span>

<br/>

<ul>
<li><span style="font-family: inherit;"><b>Haiyan Meng </b>presented </span><span style="color: #333333; font-family: inherit;"><a href="http://ccl.cse.nd.edu/research/papers/umbrella-escience-2016.pdf" target="_blank">Conducting Reproducible Research with Umbrella: Tracking, Creating, and Preserving Execution Environments</a>, which describes how the <a href="http://ccl.cse.nd.edu/software/umbrella" target="_blank">Umbrella</a> framework  is used to create a precise specification of computational environment, software, and data for reproducible execution for epidemiology and high energy physics codes.  </span></li>
<li><span style="color: #333333; font-family: inherit;"><b>Peter Ivie</b> presented </span><a href="http://ccl.cse.nd.edu/research/papers/prune-escience-2016.pdf" style="color: navy; font-family: inherit;" target="_blank">PRUNE: A Preserving Run Environment for Reproducible Computing</a><span style="font-family: inherit;"> which describes </span><a href="http://ccl.cse.nd.edu/software/prune" style="font-family: inherit;" target="_blank">PRUNE</a><span style="font-family: inherit;">, a workflow system that tracks both data and executions in a way that can be compactly named and shared.  This allows one to uniquely identify an execution in a way that others can track the complete provenance, or re-execute it if desired.</span></li>
</ul>

<span style="color: #333333; font-family: inherit;">Congrads to Haiyan for winning a </span>
