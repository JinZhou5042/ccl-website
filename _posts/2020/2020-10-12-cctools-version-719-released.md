<p>The Cooperative Computing Lab is pleased to announce the release of version 7.1.9 of the Cooperative Computing Tools including Parrot, Chirp, JX, Makeflow, WorkQueue, and other software.<br/><br/>The software may be downloaded here:<br/><a href="http://ccl.cse.nd.edu/software/download">http://ccl.cse.nd.edu/software/download</a><br/><br/>This is a bug release with some new features and bug fixes. Among them:<br/><br/></p>

<ul style="text-align: left;"><li>[Batch] Improve missing jobs detection on slurm, torque, sge, pbs. (Ben Tovar)</li><li>[Batch] WALL_TIME as a resource for slurm. (Ben Tovar)</li><li>[Makeflow] Several fixes for nested workflows. (Ben Tovar)</li><li>[Makeflow] Warn on redefinition of resources. (Ben Tovar)</li><li>[Resource Monitor] --measure-only flag when limits are specified. (Ben Tovar)</li><li>[Work Queue] API to define minimum resources for a category. (Ben Tovar)</li></ul>

<p>Thanks goes to the contributors for many features, bug fixes, and tests:<br/><br/></p>
