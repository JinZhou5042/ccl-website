The Cooperative Computing Lab is pleased to announce the release of version 6.1.6 of the Cooperative Computing Tools including Parrot, Chirp, Makeflow, WorkQueue, Umbrella, Prune, SAND, All-Pairs, Weaver, and other software.

<br/>

<br/>

The software may be downloaded here:

<br/>

<a href="http://ccl.cse.nd.edu/software/download">http://ccl.cse.nd.edu/software/download</a>

<br/>

<br/>

This is a minor release which adds some bug fixes. Among them:

<br/>

<br/>

<ul>
<li>[General] Fix bug configuring perl paths. (Ben Tovar)</li>
<li>[General] Fix bug on JX inline querying strings. (Tim Shaffer)</li>
<li>[Makeflow] Fix bug when waiting on a local process. (Douglas Thain)</li>
<li>[Makeflow] Enforce local resources limits. (Douglas Thain)</li>
<li>[Makeflow] Save failed outputs to aid debugging. (Tim Shaffer)</li>
<li>[Makeflow] Fix bug when parsing of some command lines. (Ben Tovar)</li>
<li>[WorkQueue] Transactions log adds workers disconnection reason. (Ben Tovar)</li>
<li>[WorkQueue] Fix bug when checking version of gnuplot in work_queue_graph_log. (Ben Tovar)</li>
<li>[WQmaker] Support dynamic environments. (Nicholas Hazekamp)</li>
</ul>
