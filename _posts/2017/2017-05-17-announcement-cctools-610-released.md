The Cooperative Computing Lab is pleased to announce the release of version 6.1.0 of the Cooperative Computing Tools including Parrot, Chirp, Makeflow, WorkQueue, Umbrella, Prune, SAND, All-Pairs, Weaver, and other software.

<br/>

<br/>

The software may be downloaded here:

<br/>

<a href="http://ccl.cse.nd.edu/software/download">http://ccl.cse.nd.edu/software/download</a>

<br/>

<br/>

This is a major which adds several features and bug fixes. Among them:

<br/>

<br/>

<ul>
<li>[General]  IPv4 and IPv6 mode handling fixes. (Tim Shaffer)</li>
<li>[Grow]  Fuse module for GROW-FS. (Tim Shaffer)</li>
<li>[Makeflow]  Updated manual. (Douglas Thain)</li>
<li>[Makeflow]  Support for Mesos. (Charles Zheng)</li>
<li>[Makeflow]  Support for Singularity. (Kyle Sweeney)</li>
<li>[Makeflow]  --shared-fs option. (Nick Hazekamp)</li>
<li>[Makeflow]  --preserve option for per rule caching. (Pierce Cunneen)</li>
<li>[Parrot]  Fix ld.so and exec bugs. (Tim Shaffer)</li>
<li>[Parrot]  Fix handling of namespace symlinks. (Tim Shaffer)</li>
<li>[Parrot]  Bind/connect on AF_UNIX sockets. (Tim Shaffer)</li>
<li>[Parrot]  Several bug fixes. (Tim Shaffer)</li>
<li>[Parrot]  parrot_namespace to use parrot inside parrot. (Tim Shaffer)</li>
<li>[Parrot]  Add fixed and warped PIDs. (Douglas Thain)</li>
<li>[Prune]     Several bug fixes. (Peter Ivie)</li>
<li>[ResourceMonitor]  Resource per task snapshots, --snapshot-file. (Ben Tovar)</li>
<li>[Umbrella]  Several bug fixes. (Haiyan Meng)</li>
<li>[WorkQueue]  Resource per task snapshots, q.enable_monitoring_snapshots. (Ben Tovar)</li>
<li>[WorkQueue]  Several bug fixes. (Ben Tovar)</li>
<li>[WorkQueue]  Custom environments for wq factory. (Kyle Sweeney)</li>
<li>[WorkQueue]  Several fixes to wq factory. (Nate Kremer-Herman)</li>
<li>[WQ_Maker]   Several bug fixes, update to latest version of maker. (Nick Hazekamp)</li>
</ul>
